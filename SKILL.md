# oh-deploy-wizard

OpenClaw 一键部署助手。简化安装、配置和监控流程。

## 功能特性

- 🚀 一键部署：自动化安装和配置
- 🔧 智能配置：交互式设置向导
- 📊 健康监控：实时状态检查
- 🐳 Docker 支持：容器化部署
- 🔄 自动更新：保持最新版本
- 🛡️ 安全加固：自动应用最佳实践

## 使用方法

### 部署 OpenClaw

```
部署 OpenClaw
```

### 检查健康状态

```
检查 OpenClaw 健康状态
```

### 更新

```
更新 OpenClaw
```

### 备份配置

```
备份 OpenClaw 配置
```

## 部署类型

- **本地**：直接安装在机器上
- **Docker**：容器化部署
- **云端**：VPS 部署

## 配置向导步骤

1. 系统要求检查
2. 选择部署类型
3. 安装 OpenClaw
4. 配置通道
5. 设置模型
6. 安全加固

## 健康检查项

- Gateway 状态
- Node 连接
- 通道连接
- 模型可用性
- 资源使用

## 依赖

- Node.js >= 18
- npm >= 9
- 最少 2GB 内存
